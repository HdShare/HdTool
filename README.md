### 介绍:

- 隐藏Root、Xp
- 隐藏Vpn、Wifi代理
- 伪装版本号、版本名
- 弹窗 强制可取消+拦截
- 事件监听(单击、长按、文字、退出)
- 算法监听(Base64、Md5等)
- Json监听
- 文件监听(读、写、删)
- 签名监听
- Shell命令监听
- 解除截屏限制
- 锁机软件监听输入框密码
- 广告屏蔽(穿山甲、优量汇、米盟)
- ACE警告
- CloudInject1.7.6弹窗
- Arm Pro1.3.1弹窗
- 小纸片2.0弹窗
- Andlua监听
- GG修改器监听
- 以及一大堆没有写的功能
